2. Перечислите плюсы и минусы SPA приложений?

плюсы:
- универсальность (применим и для десктопа, и для веба, и отчасти для мобильных приложений)
- размер приложения и используемых им данных не ограничен памятью устройства
- высокая скорость работы сайта/приложения
- удобство за счет набора готовых элементов для работы

минусы:
- обязателен доступ к интернету
- трудности с SEO
- не работает у пользователей с отключенной поддержкой JS

3. Как создать React-приложение?

1) Инициализировать папку как npm-проект с помощью npm init -y
2) Установить node-пакеты React npm install react react-dom + babel (или npm install -g create-react-app)
3) Инициализировать приложение create-react-app firstapp

ИЛИ ввести комманду npx create-react-app название приложения
